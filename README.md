## 🛡️ Infrastructure Resilience & Autonomous Protection

### 6-State Finite State Machine (FSM)

```
State 0 → BOOT        (system initialization & health check)
State 1 → ACTIVE      (normal full-capacity operation)
State 2 → DEGRADED    (reduced capacity, fault isolation)
State 3 → SAFE_MODE   (capital deployment suspended)
State 4 → EMERGENCY   (market-close override triggered)
State 5 → RECOVERY    (autonomous restart & state validation)

Responsibilities:
├── Lifecycle supervision
├── Failure-state isolation
├── Emergency degradation handling
├── Autonomous recovery orchestration
└── Infrastructure health-state monitoring
```

```
Operation:  Physically separated from primary execution host

Monitors:
```

## ⚙️ Getting Started

### Prerequisites

```
Python 3.11+
Rust (nightly toolchain)
Ollama (local inference server)
```

### Installation

```bash
# Clone the repository
git clone https://github.com/GaneshEiGo/EiFate-OMEGA-SUPREME.git
cd EiFate-OMEGA-SUPREME

# Python environment
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# Rust execution engine
cargo build --release

# Pull local inference models
ollama pull qwen2.5:7b
ollama pull qwen2.5:32b
ollama pull llama3:8b
```

### Configuration

```yaml
# config/system.yaml
system:
  name: EiFate-OMEGA-SUPREME-V2026.1
  mode: autonomous
  operation: 24_7

engines:
  grinder:
    concurrent_mechanisms: 12
    enabled: true
  leviathan:
    confidence_gates: 7
    enabled: true
  phantom:
    obfuscation: true
    twamm_enabled: true
  oracle:
    sentiment_refresh_ms: 200
    macro_stream: true

execution:
  hot_path_latency_ms: 50
  hw_verification: true
  rollback_on_failure: true

validation:
  pre_trade_gates: 8
  p_value_threshold: 0.05
  bonferroni_correction: true

resilience:
  watchdog_heartbeat_ms: 500
  fsm_states: 6
  spore_reserve_percent: 20
  emergency_override: true
```

### Launch

```bash
# Start the full autonomous system
python main.py --mode live

# Run in simulation mode
python main.py --mode simulation

# Engine-level diagnostics
python main.py --engine grinder --diagnostic
python main.py --engine oracle --diagnostic

# Validation suite
pytest tests/ -v
```
